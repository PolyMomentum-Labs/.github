# PolyMomentum · Organization Profile

This repository powers the **PolyMomentum** GitHub Organization profile and shared community documentation.

## What this repo contains

| Path | Purpose |
|------|---------|
| `.github/profile/README.md` | Organization homepage content (rendered on [github.com/PolyMomentum](https://github.com/PolyMomentum)) |
| `CONTRIBUTING.md` | Contribution guidelines for org repositories |
| `SECURITY.md` | Vulnerability reporting policy |
| `CODE_OF_CONDUCT.md` | Community standards |

## Publishing

1. Create or use the **`.github`** repository under the [PolyMomentum](https://github.com/PolyMomentum) organization.
2. Push the contents of this folder to that repository.
3. GitHub will automatically render `.github/profile/README.md` on the organization homepage.
